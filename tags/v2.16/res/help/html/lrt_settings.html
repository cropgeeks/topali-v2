LRT Settings

LRT stands for Likelihood Ratio Test.

Please ensure that at least 3 sequences have been selected from your alignment before attempting to run LRT. See selecting sequences for further information.

To run the LRT analysis method, select Analysis | Run LRT from the menu bar. This brings up the Settings dialog that provides two groups of parameters that can be customized before starting the LRT run.

Basic settings:
Basic settings are available by selecting the Basic tabbed pane.

  • Step size - step size (in nucleotides) to use when moving along the alignment.
  • Window size - window size (in nucleotides) to use when analyzing along the alignment.
  • Analysis run method - specifies the type of LRT analysis to be done

Advanced settings
Advanced settings are available by selecting the Advanced tabbed pane. It is advisable not to change these settings unless you understand the methodology behind LRT.

  • Number of bootstrapping threshold runs - number of times to re-run the analysis to calculate the threshold values.
  • Nucleotide substitution model - nucleotide substitution model to use in distance matrix calculations.

Once you are happy with the settings, click Run to begin the analysis.
