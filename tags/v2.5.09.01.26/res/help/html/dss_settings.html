DSS Settings

DSS stands for Difference of Sums of Squares.

Please ensure that at least 3 sequences have been selected from your alignment before attempting to run DSS. See selecting sequences for further information.

To run the DSS analysis method, select Analysis | Run DSS from the menu bar. This brings up the Settings dialog that provides two groups of parameters that can be customized before starting the DSS run.

Basic settings:
Basic settings are available by selecting the Basic tabbed pane.

  • Step size - step size (in nucleotides) to use when moving along the alignment.
  • Window size - window size (in nucleotides) to use when analyzing along the alignment.
  • Analysis run method - specifies the type of DSS analysis to be done

Advanced settings
Advanced settings are available by selecting the Advanced tabbed pane. It is advisable not to change these settings unless you understand the methodology behind DSS.

  • Number of bootstrapping threshold runs - number of times to re-run the analysis to calculate the threshold values.
  • Least squares method - least squares method to use when calculating sum of squares.
  • Nucleotide substitution model - nucleotide substitution model to use in distance matrix calculations.
  • Number of passes - number of passes to perform over the alignment (one: forward only, two: forward and backward).

Once you are happy with the settings, click Run to begin the analysis.
