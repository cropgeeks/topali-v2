Auto Partitioning

Alignment partitions can be created automatically, based on the threshold levels of existing (compatible) analysis results sets.

To do this, select Analysis | Auto-Partition Alignment from the menu bar, or click the Auto Partition button on the Current Partitions dialog. Either option will bring up the Auto-Partition Alignment dialog.

This lists all results that can be partitioned (those that contain significance threshold lines). The result set chosen will be the set used to break up the alignment based on the midpoints between points that cut the threshold. As very small partitions may be meaningless, an option exists to discard partitions whose length is less than the chosen value.
