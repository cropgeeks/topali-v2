Finding Sequences

To locate and display a particular sequence within an alignment, select Alignment | Find Sequence from the menu bar. This brings up the Find Sequence dialog.

To search for a sequence, enter its exact name into the text box. Two additional options are also provided to help customize the search:

  • Select this sequence if it is found - ensures that the sequence being searching for becomes the only selected sequence.
  • Match case - perform a case-sensitive name search.
