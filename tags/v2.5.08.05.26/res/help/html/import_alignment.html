Importing Alignments

To import an alignment into TOPALi, select File | Import Alignment from the menu bar. This brings up the Import Alignment dialog.

From here, you can choose to either import an alignment directly, or to import an alignment by first creating one by using unaligned cDNA and a guide protein alignment.

TOPALi can currently import alignments that are in formats supported by the popular ReadSeq program. Basic, FASTA formatted files are loaded directly by TOPALi - all other formats are handed off to an external ReadSeq web service to be converted into a format that TOPALi does understand.

If you would like support added for an alignment file format that TOPALi can not handle, then please get in touch with us at topali (at) bioss.ac.uk
