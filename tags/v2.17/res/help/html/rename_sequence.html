Renaming Sequences

To rename a sequence, first select it from the sequence list with the mouse, then select Alignment | Rename Sequence from the menu bar. This brings up the Rename Sequence dialog. You can then enter a new name for the sequence.

Note that TOPALi will only accept names that are:

  • Unique within the current alignment.
  • Do not contain space characters.
  • Do not contain characters (such as parentheses) that could cause problems if the sequence is exported to an external application.
