Selecting Sequences

When an alignment is first imported into TOPALi, all of its sequences are selected by default. However, certain tasks (such as the creation of phylogenetic trees, the recombination detection methods, etc) may require an exact number of sequences to be selected that differs from the number contained in the alignment. You may also wish to only analyze certain sequences.

Manually selecting sequences:
To select sequences, click their names with the mouse.

  • A single click will select only that sequence.
  • Holding CTRL while selecting will select each sequence in turn.
  • Holding SHIFT while selecting will select a group of sequences.

Automatically selecting sequences
You can also select groups of sequences using the Alignment | Select Sequences menu options. Here you can:

  • Select all sequences.
  • Deselect all sequences (select none).
  • Select only those sequences that are unique.
  • Invert the current selection (selected become unselected and vice versa).
