Importing Alignments

To import an alignment into TOPALi, select File | Import Alignment from the menu bar, then select the file to import using the File Chooser dialog that appears.

TOPALi can currently import alignments that are in the following formats:

  • fasta
  • phylip (sequential
  • phylip (interleaved)
  • nexus (standard)
  • nexus (MrBayes)
  • GCG/MSF
  • clustalW

If you would like support added for an alignment file format not listed above then please get in touch with us at topali (at) bioss.ac.uk
