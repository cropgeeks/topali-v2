TOPALi v2

TOPALi v2 provides an intuitive graphical interface for working with existing statistical methods designed for detecting recombination in DNA sequence alignments. "Version 2" of TOPALi is a complete overhaul to the original TOPALi with most of its analysis methods reimplemented as web services for parallel computing on remote high-performance clusters.

Current release is 2.05 9th Mar 2006 (view changelog)

(Note that release 2.05 is not compatible with projects created with earlier versions)
