KNOWLEDGE TRANSFER : TOPALi v2

TOPALi v2 provides an intuitive graphical interface for working with existing statistical methods designed for detecting recombination in DNA sequence alignments. "Version 2" of TOPALi is a complete overhaul to the original TOPALi with most of its analysis methods reimplemented as web services for parallel computing on remote high-performance clusters. TOPALi is also multi-core aware, so if you have more than one CPU (or CPU core) in your PC then TOPALi can take advantage of them when running calculations locally.
