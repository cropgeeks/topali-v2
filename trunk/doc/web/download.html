KNOWLEDGE TRANSFER : TOPALi v2

Download TOPALi

If you already have Java 1.5.x (or later) installed, you may be able to run TOPALi by using Java Web Start. This ensures you always have the latest version of the software.

  • Run TOPALi v2 with Java Web Start

If you are unable to run the above, or you would rather download a version of TOPALi that can be installed, then please choose the most appropriate installer for your system from the list below. Note that these installers will download a suitable version of Java for you if your machine does not already have it.

  • Windows Installer
  • Linux Installer
  • Solaris Installer
  • Mac OS X Installer (untested)
  • Binaries (required for running jobs locally)
  • Source Code (complete with Ant build scripts)

Note: if you are upgrading from an earlier version of TOPALi, there is no need to uninstall your current version before upgrading. Simply install the new version to the same location as your current version, and all your settings will be maintained.
