Estimating Phylogenetic Trees

Phylogenetic trees can be created from any selected partition, and from any set of selected sequences (where the number of sequences is greater than three).

To create a phylogenetic tree, select Analysis | Estimate Phylogenetic Tree from the menu bar. This brings up the Estimate New Tree dialog.
