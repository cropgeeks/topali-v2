Partitioning

Working with partitions involves the use of the Current Partitions dialog, accessible by selecting Alignment | Show Partitions Dialog from the menu bar.

A partition is defined as any region of the current alignment, and contains both a starting nucleotide and an ending nucleotide. Splitting an alignment up into multiple partitions allows you to more easily analyze the various regions, and perform additional tasks upon them such as estimating phylogenetic trees relate only to that region and not the alignment as a whole.

The Current Partitions dialog lists all defined partitions for an alignment, and also provides a Jukes Cantor/Neighbor Joining phylogenetic tree preview of the currently selected partition.

Working with partitions:
The dialog contains several buttons for working with partitions:

  • New - manually specific a new partition.
  • Edit - edit selected partition.
  • Auto Partition - automatically create partitions based on analysis results.
  • Remove - remove selected partition from list.
  • Remove All - remove all partitions from list.

Two shortcut buttons for other commonly used TOPALi functions are located here too:

  • Export - export selected partition to disk.
  • Create Tree - estimate phylogenetic tree from partition.

Viewing partitions:
In addition to the JC/NJ tree preview, the currently selected partition will also be mirrored in TOPALi's Alignment View by dimming regions of the alignment outwith the selected partition's start and end points. Similarly, result sets that show a graph plotted against alignment length will highlight the current partition in a lighter colour.

Selecting partitions:
In addition to creating partitions manually, it is also possible to create a new partition by selecting it directly from compatible graphs.

To define a new partition click and hold the left-mouse button over the graph, then drag left or right with the mouse until the partition highlighting covers the desired region. Release the mouse to confirm the new partition.

Note that partitions created in this way are not automatically added to the Current Partitions dialog. To do this, click the Add Selection as New Partition button located on the graph's toolbar.
