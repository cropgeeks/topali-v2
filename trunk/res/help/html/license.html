External application

PDM analysis requires the use of BAMBE and TreeDist
   http://www.mathcs.duq.edu/larget/bambe.html
   http://evolution.genetics.washington.edu/phylip.html

DSS analysis requires the use of Fitch
   http://evolution.genetics.washington.edu/phylip.html

Model Selection is based on ModelGenerator
   http://bioinf.may.ie/software/modelgenerator/

Bayesian tree estimation is based on MrBayes
   http://mrbayes.csit.fsu.edu/

Maximum likelihood tree estimation is based on PhyML
   http://atgc.lirmm.fr/phyml/

Ancestral sequence estimation is based on FastML
   http://www.tau.ac.il/~talp/supplementary/fastml/fastml.2.02.htm

Positive Selection Analysis is based on PAML
   http://abacus.gene.ucl.ac.uk/software/paml.html

Codon Usage Analysis is based on CodonW
   http://codonw.sourceforge.net/

BAMBE, Fitch and TreeDist are all included as a part of TOPALi with their respective authors' prior permission.
