Import an alignment by selecting File | Import Alignment from the menu bar.

Select which sequences you wish to analyze by clicking the them with the left mouse button. Use ctrl+click or shift+click to select groups of sequences.

Run one of the three recombination detection algorithms by selecting it from the Analysis menu. HMM is suitable for 4 sequences only, PDM is suitable for 4-10 sequences and DSS for 4+ sequences.

Choose suitable window and step sizes for the method. Too big a window and the method might miss likely recombination, too small a window and the results will be too noisy.
