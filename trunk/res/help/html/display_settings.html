Display settings

A range of visual options can be customized by selecting View | Display Settings from the menu bar. This brings up the Display Settings dialog.

To change the settings for colours, simply select a colour with the mouse. This brings up a standard Colour Chooser dialog where you can then select a new colour to use instead. Any changes made will take effect immediately, with the main window updating to reflect the changes even while the dialog is still visible.

A range of additional options are also available:

  • Font size - specifies the size used to draw nucleotide data.
  • Colour seed - specifies the seed used to generate random colours for graphs, trees and histograms.
  • Show nucleotides on display - toggle displaying nucleotide data on the sequence display area.
  • Show colour highlighting - toggle the highlighting of nucleotide data with background colours.
  • Dim unselected sequences - toggle drawing unselected sequences in a dim style.
  • Use bold font - toggle displaying nucleotide data in a bold font.
  • Show Sequences pane tooltips - toogle showing sequence and nucleotide information as a tooltip over the Sequences pane
  • Vertical Highlight - toggle highlighting nucleotides below the cursor.
  • Horizontal Highlight - toggle highlighting sequence below the cursor.

To reset all options back to their default values, click the Defaults button.
