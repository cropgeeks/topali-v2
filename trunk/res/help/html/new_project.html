Creating a new project

To create a new project, select File | New Project from the menu bar, click the New Project toolbar button, or press Ctrl-N. This brings up the New Project Wizard dialog.

     New Project

Project creation

The first stage in creating a new project, is to choose a name (for the project), and a location on disk where it will be stored.

  • Name - name of the new project.
  • Location - location on disk where the project will be stored.
  • Browse - browse for and select a project location.

Click Next once a name and a location have been picked.

 

Select alignment

The second stage is to select an alignment file to work with. Note that the new project will only contain a copy of the existing alignment, meaning any changes made to it from within TOPALi will not affect the original.

  • Alignment - location on disk of the alignment file to use with this project.
  • Browse - browse for and select an alignment file.

Click Next once a file has been selected, and the project will then be created, and your alignment file loaded into it.
