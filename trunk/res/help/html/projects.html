Projects

TOPALi stores all user data and results in projects files (with the extension .topali).

When TOPALi first loads, a default, empty project is created automatically, which allows you to import alignments and begin working straight away. It is only after you attempt to save your work that you will be prompted for a filename for this project.

To create a new project:
Select File | New Project from the menu bar.

To save an existing project:
Select File | Save Project or File | Save Project As from the menu bar, the latter option allowing you to save the current project under a different filename.

To open an existing project:
Select File | Open Project from the menu bar, or select from the list of recently opened projects using the File | Recent Projects sub menu.

A TOPALi project file stores:

  • Imported alignments.
  • Currently selected sequence data.
  • Generated phylogenetic trees and their display settings.
  • All generated analysis results (and the settings used to obtain them).
  • Partition information.
