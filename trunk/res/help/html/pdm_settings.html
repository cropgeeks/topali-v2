PDM Settings

PDM stands for Probabilistic Divergence Measures.

Please ensure that at least 4 sequences have been selected from your alignment before attempting to run PDM. See selecting sequences for further information.

To run the PDM analysis method, select Analysis | Run PDM from the menu bar. This brings up the Settings dialog that provides two groups of parameters that can be customized before starting the DSS run.

Basic settings:
Basic settings are available by selecting the Basic tabbed pane.

  • Step size - step size (in nucleotides) to use when moving along the alignment.
  • Window size - window size (in nucleotides) to use when analyzing along the alignment.

Advanced settings
Advanced settings are available by selecting the Advanced tabbed pane. It is advisable not to change these settings unless you understand the methodology behind PDM.

  • Number of bootstrapping threshold runs - number of times to re-run the analysis to calculate the threshold values.
  • Prune histogram results - toggle whether or not to use Pruned PDM (recommended for large numbers of sequences).
  • Seed - 1 < odd integer < 2147483647 (2^31 - 1).
  • Burn-in time - number of cycles to run burn algorithm.
  • Cycles - number of cycles to run main algorithm.

  • Use-beta - use scaled beta distribution modification of local.
  • Molecular-clock - if true, assume constant rate of evolution for all branches.
  • Category-list - one site category.

  • Parameter-update-interval - number of tree updates after each parameter update.
  • Update-theta - substitution rate parameters.
  • Tune-interval - interval to adapt to global-tune during burn-in.
  • Global-tune - half window width for global.
  • Local-tune - stretch parameter for local.
  • Theta-tune - Dirichlet parameter for theta update.
