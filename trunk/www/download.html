TOPALi is compatible with Microsoft Windows, Apple Mac OS X, Linux, and Solaris. Please choose the most appropriate installer for your system from the list below.

   Windows Installer

Note: if you are upgrading from an earlier version of TOPALi, there is no need to uninstall your current version before upgrading. Simply install the new version to the same location as your current version, and all your settings will be maintained.

The source code for TOPALi is publically available via Subversion at the following URL: svn://subversion.scri.ac.uk/sbrn/projects/topali-v2

 

TOPALi is developed on Microsoft Windows XP SP3, and gets additional testing on OS X 10.5.4, Ubuntu Linux 7.10, and Red Hat/Centos 5.2. Other systems are obviously untested, so please let us know of problems if you encounter them.
